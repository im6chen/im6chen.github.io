V2RAY

安装

apt update

apt upgrade

apt install curl

apt install vim

curl -O https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh

安装&更新

bash install-release.sh

配置

配置文件路径 /usr/local/etc/v2ray/config.json

uuid: 3a76edb4-0bde-4ec7-9df6-19abefa7e50b

最好还是使用 V2Ray 提供的配置检查功能（test 选项），因为可以检查 JSON 语法错误外的问题，比如说突然间手抖把 vmess 写成了 vmss，一下子就检查出来了。

v2ray -test -config config.json
