树莓派初始化

Author: im6chen Date: 2021-9-23  10:53 Category: linux

1. 刷系统

Raspberry Pi Imager写入系统

2. 添加ssh和config文件

3. MAC 删除ssh key

终端输入：vi ~/.ssh/known_hosts

删除内容 : 保存退出

4. 可以ssh到以后，换源

https://mirror.tuna.tsinghua.edu.cn/help/raspbian/

5. 打开vnc

进入后选择Interfacing Options>>VNC>>Yes
